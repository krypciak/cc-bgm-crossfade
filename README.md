# cc-bgm-crossover
Adds API for smoothly switching between the regular and battle background music

## Building

```bash
git clone https://github.com/krypciak/cc-bgm-crossover
cd cc-bgm-crossover
pnpm install
pnpm run start
# this should return no errors (hopefully)
npx tsc
```
